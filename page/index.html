What's in the box ?

Moleskin is a light WYSIWYG, with almost no dependencies (only two files). Angular and jQuery compatible.

  • Only 35kb minified
  • Markdown to HTML
  • HTML to Markdown
  • Autogrow feature

Quick start

Angular

<!-- JS --> <script type="text/javascript" src="./jquery-1.9.1.min.js"></script> <script type="text/javascript" src="./dist/angular-moleskin.min.js"></script> <!-- CSS --> <link rel="stylesheet" href="./moleskine.css" /> <moleskin bind-data="content2" base-content="static_content" class="hey" width="550" css-class="mole" default-mode="markdown" input="markdown" output="html" height="450"></moleskin>

jQuery

<!-- JS --> <script type="text/javascript" src="./jquery-1.9.1.min.js"></script> <script type="text/javascript" src="./dist/moleskin.min.js"></script> <!-- CSS --> <link rel="stylesheet" href="./moleskine.css" /> <script type="text/javascript"> $(document).ready(function() { el = $('.rte1').moleskine({ width : 800, height : 500, defaultMode : 'markdown', onChange : function(err, data) { console.log(data); } }); }); </script>
